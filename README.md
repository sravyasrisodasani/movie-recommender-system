# 🎬 Movie Recommender System

A content-based movie recommendation system built using **Python, Machine Learning, and Streamlit**.

## 🚀 Features
- Recommends movies based on similarity
- Built using TMDB dataset
- Uses Cosine Similarity for recommendations
- Interactive web interface with Streamlit

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- NLTK

## 📂 Project Structure
app.py → Streamlit app
movies_dict.pkl → movies
similarity.pkl → similarity matrix

## ▶️ Run the App

Install dependencies:

pip install pandas scikit-learn streamlit nltk

Run the app:

streamlit run app.py
